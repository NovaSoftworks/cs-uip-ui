# cs-uip-ui

This repository contains the User Identity Platform user interface.

## Getting Started

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-org/cs-uip-ui.git
    cd cs-uip-ui
    ```

2. **Install dependencies:**
    ```bash
    npm install
    ```

3. **Run the development server:**
    ```bash
    npm run dev -- --open
    ```
